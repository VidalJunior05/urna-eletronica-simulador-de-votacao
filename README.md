## 📁 How to download the Project 
```bash 
    #clone the repository
     $ git clone https://github.com/VidalJunior05/urna-eletronica-simulador-de-votacao.git

    # Enter directory
    $ cd urna_eletronica

    # install dependencies
    $ yarn install

    # Initialize project
    $ yarn start
```
